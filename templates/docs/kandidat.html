Kelola Kandidat

Kandidat adalah orang yang melamar atau diprospek untuk posisi di perusahaan. Satu kandidat bisa punya banyak pipeline (melamar ke beberapa posisi sekaligus).

Sumber Kandidat

Ada tiga cara menambah kandidat:

  • Import NocoDB — tarik data dari database NocoDB yang terhubung
  • Entry Manual — isi form nama, email, phone, posisi saat ini, CV link
  • Dari Job Detail — tambah langsung saat membuka pipeline baru

Halaman Daftar Kandidat

Tabel kandidat mendukung pencarian lanjutan:

  • Search — nama atau email
  • Filter — bahasa, cloud, tools, posisi, arrangement, notice period

Profil Kandidat

Klik kandidat untuk melihat profil lengkap:

  • Data personal (nama, email, phone, posisi, YOE)
  • Skills (languages, cloud, tools)
  • Preferensi (working arrangement, notice period, salary)
  • CV link
  • Daftar pipeline aktif
  • Riwayat komentar dan aktivitas
