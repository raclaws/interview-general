Mulai di Sini

Halaman ini menjelaskan alur kerja rekrutmen dari awal sampai akhir menggunakan INS ATS. Ikuti langkah-langkah di bawah ini untuk pertama kali menggunakan sistem.

Alur Kerja Rekrutmen

  1. Buat Business Unit — masuk ke Settings → Business Units, tambahkan divisi (misal: Engineering, Product).
  2. Buat Lowongan (Job) — dari halaman Jobs, klik "+" untuk membuat lowongan baru. Isi posisi, level, recruiter, dan headcount.
  3. Tambah Kandidat — import dari NocoDB, entry manual, atau dari halaman Candidates.
  4. Masukkan ke Pipeline — dari Job detail, tambahkan kandidat ke pipeline. Kandidat akan mulai di tahap "Applied".
  5. Buat Sesi Wawancara — dari Pipeline detail, buat session dan kirim link ke interviewer.
  6. Pindahkan Tahap — setelah wawancara selesai, geser pipeline stage sesuai keputusan (interview → offered → hired).
  7. Generate Offer — jika kandidat lolos, generate surat offering dari Pipeline detail.

Login Pertama Kali

Buka browser dan akses URL aplikasi. Anda akan melihat halaman login.

Belum punya akun admin? Jalankan python -m app.cli create-admin di terminal server untuk membuat akun pertama.

Keyboard Shortcut

INS ATS mendukung navigasi keyboard untuk produktivitas:

  • j / k — navigasi baris atas/bawah di tabel
  • Enter — buka item yang sedang di-highlight
  • Backspace — kembali ke halaman sebelumnya
  • / — fokus ke search bar
  • Ctrl+Z — undo delete terakhir
