MCP (Agent Tools)

INS ATS menyediakan server MCP (Model Context Protocol) agar AI agent bisa mengakses data rekrutmen. Berguna untuk automasi, reporting, dan integrasi.

Menjalankan MCP Server

python -m app.mcp_server

Server berjalan via SSE transport. AI agent (Claude, dll) bisa konek dan memanggil tools.

Tools Tersedia

Sessions

create_sessionBuat session baru + interviewer links
get_sessionDetail session (skor, status, summary)
list_sessionsDaftar session (filter by candidate)

Jobs

list_jobsDaftar lowongan (filter: status, BU)
get_jobDetail job + pipeline list

Pipelines

list_pipelinesDaftar pipeline (filter: job, stage)
get_pipelineDetail pipeline + candidate + sessions

Candidates

list_candidatesCari kandidat (nama/email, max 50)
get_candidateDetail kandidat + pipeline aktif

Tasks

list_tasksDaftar tugas (filter: entity, status)
create_taskBuat tugas baru di job/pipeline
update_taskUpdate status/title/priority/due date

Reports

generate_recruitment_reportGeneral report (filter: BU, level, period)
generate_pipeline_reportPipeline assessment report
generate_job_reportJob health report

Contoh Penggunaan

AI agent bisa diminta:

  • "Buatkan laporan rekrutmen Engineering bulan ini"
  • "List semua pipeline yang masih di stage interview"
  • "Tambahkan task 'Follow up offering' ke pipeline kandidat X"
  • "Berapa headcount yang sudah terpenuhi untuk job Backend Developer?"
MCP server menggunakan database yang sama dengan web app — data selalu up to date.
