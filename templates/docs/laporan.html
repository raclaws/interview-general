Laporan

INS ATS bisa membuat laporan otomatis menggunakan AI. Laporan di-generate sebagai file HTML standalone yang bisa di-download atau dibagikan.

Jenis Laporan

Tipe Deskripsi Filter
GeneralRingkasan rekrutmen keseluruhanBU, level, periode
JobHealth report per lowonganPeriode
PipelineAssessment report per kandidat—

Membuat Laporan

  1. Buka halaman Reports dari sidebar.
  2. Pilih tipe laporan.
  3. Atur filter jika diperlukan (business unit, level, periode).
  4. Klik Generate — proses memakan 10-30 detik (AI processing).
  5. Laporan siap di-download atau dilihat langsung di browser.
