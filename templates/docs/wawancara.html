Wawancara & Skor

Sistem wawancara INS ATS menggunakan token link — admin membuat session, interviewer mengisi form lewat link unik tanpa perlu login.

Membuat Sesi Wawancara

  1. Buka halaman detail pipeline kandidat.
  2. Di section Sessions, klik + New Session.
  3. Pilih template wawancara (HR Interview, Culture Alignment, atau custom).
  4. Tambahkan nama interviewer (bisa lebih dari satu).
  5. Klik Create.

Mengirim Link ke Interviewer

Setelah session dibuat, setiap interviewer mendapat link unik. Cara menyalin:

  • Dari halaman Interview (sidebar) — klik icon copy di baris session
  • Dari Session detail — klik link interviewer
Link bersifat single-use. Setelah interviewer submit form, link tidak bisa digunakan lagi.

Form Interviewer

Interviewer membuka link dan melihat form sesuai template — berisi pertanyaan per dimensi dengan skala 1-4 dan kolom free text.

Scorecard Otomatis

Setelah semua interviewer submit, scorecard otomatis ter-generate di pipeline detail:

  • HR Dimensions (max 12) — Ownership, Maturity, Collaborative
  • Culture Dimensions (max 16) — Execution, Learn Fast, Impact, Clarity
  • Per-interviewer breakdown dan rata-rata

LLM Summary

Setelah semua response masuk, admin bisa klik Generate Summary untuk mendapatkan rangkuman AI dari seluruh jawaban — termasuk cross-evaluator insight jika ada lebih dari satu interviewer.
